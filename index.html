What is the use of preformatted text in HTML ?

The 'pre' (preformatted) element is used to define a block of text to diplay them in the same manner as is writtenin the HTML file.

<pre> represents preformatted text,
     preserving      all spaces,
new lines, and formatting as written in the HTML code.
